Bento Editor Approaches

How should users interact with the configuration tool?

1

The "Live Studio"

RECOMMENDED

A dedicated /studio route. Features a side panel for widgets and a main area for the grid. A "Save" button persists changes to the file system.

2

The "Sandbox Builder"

A standalone builder. It doesn't touch your files. Instead, it generates a "Download lobby.json" or "Copy to Clipboard" button at the end.

3

The "Command Panel"

Edit directly on the homepage. Toggle "Edit Mode" to unlock cards for dragging and resizing. Feels the most modern but is harder to implement perfectly.
